# Fusion Calculator
A calculator for various fusions and combinations. Planned to be usable for any kind of fusions in the future via **Lua** API, for external/additional fusions
 * Official fusions will be done in **C++**

## Planned fusion calculations
 * Persona fusions
 * SMT fusions
